# 课程概述 #

培训将介绍新的概念和术语，并帮助学生熟悉自上次发布以来发生变化的方法。


## 课程结构 ##

该课程由不同部分组成，包括：

- 2018的概述
- 使用“部分运行”工具创作工作流程
- 可折叠书签和相关功能
- 空间变换的精度和容差
- 写入和样式化Microsoft数据格式
- 使用新的FeatureJoiner转换器
 


## 关于手册 ##
升级到FME Desktop 2018培训手册不仅构成了本培训课程的基础，而且也是您未来与FME合作的有用参考资料。

这些材料中的所有屏幕截图都是在Windows Server 2016上使用FME拍摄的。使用的字体（特别是在日志窗口的屏幕截图中）可能会调整大小或以其他方式更改，以提高可读性。
