# 桌面基础训练手册 #

这是安全软件FME桌面应用入门级培训课程的手册。

![](./DesktopBasic0Introduction/Images/Img0.0.FMEAboutScreen.png)

- 数据转换基础
- 数据转换
- 工作空间设计
- 实际转换器使用
- 最佳实践
