# 升级到FME Desktop 2018培训课程 #

这是用于升级到Safe Software的FME Desktop应用程序的2018版本的用户的培训手册。

![](./2018Upgrade0Introduction/Images/Img0.0.FMEAboutScreen.png)

培训将介绍新的概念和术语，并帮助学生熟悉自上次发布以来发生变化的方法。


## 课程结构 ##

该课程由不同的部分组成，包括：

- 2018的概述
- 使用“部分运行”工具创作工作流程
- 可折叠书签和相关功能
- 空间变换的精度和容差
- 写入和样式化Microsoft数据格式
- 使用新的FeatureJoiner转换器
