# FME Server管理培训手册 #

这是FME Server系统管理培训课程的手册。

![](./ServerAdmin0Introduction/Images/0.000.ServerHomeScreen.png)

## 课程结构 ##

全部课程由七个部分组成。这些部分是：

- 规划FME Server安装
- FME Server连通性
- FME Server安全性
- 可扩展性和性能
- FME服务器自定义
- 迁移和升级
- 故障排除
